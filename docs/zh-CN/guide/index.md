# 入门指南

本章节包含 SafeMySQLMcpServer 的入门教程和使用指南。

## 内容

| 指南 | 说明 | 预计时间 |
|------|-------------|------|
| [快速开始](./getting-started.md) | 安装并运行 SafeMySQLMcpServer | 15 分钟 |
| [认证配置](./authentication.md) | 设置 JWT 认证 | 5 分钟 |
| [安全配置](./security-config.md) | 配置 SQL 安全规则 | 10 分钟 |
| [集成 Claude Code](./claude-code.md) | 连接 Claude Code 到数据库 | 10 分钟 |

## 前置条件

- Go 1.22 或更高版本
- MySQL 5.7 或更高版本（或 MariaDB 10.2+）
- Docker（可选，用于容器化部署）
