## 查询工具数据流
```
1. 客户端发送: { "tool": "query", "database": "mydb", "sql": "SELECT * FROM users" }
                              │
                              ▼
2. JWT 中间件: 从 token 提取 user_id, user_email → 添加到 context
                              │
                              ▼
3. 速率限制器: 检查基于 IP 的速率限制 (默认: 100 请求/分钟)
                              │
                              ▼
4. MCP Handler:
   a. 验证数据库名称: "mydb" → 正则检查
   b. 验证 SQL: "SELECT * FROM users" → 长度检查
   c. 解析 SQL: vitess 解析器 → AST
   d. 安全检查: SELECT 是否允许? → 是
   e. 从路由器获取数据库连接
   f. 执行查询（带超时）
   g. 收集结果（最多 10000 行）
                              │
                              ▼
5. 审计日志记录: { user_id, database, sql, status, duration_ms, rows_affected }
                              │
                              ▼
6. 响应: { "columns": [...], "rows": [...], "rows_affected": N }
```
